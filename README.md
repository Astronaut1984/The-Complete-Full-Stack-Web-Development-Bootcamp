## The Complete Full-Stack Web Development Bootcamp

This repository documents my work for "The Complete Full-Stack Web Development Bootcamp" course by Angela Yu. It contains notes, exercises, examples, solutions, and small projects created while following the course.

Purpose

- Document progress, examples, and solutions from the course.
- Keep course files organized by section for easy review and reference.

How to use this repo

- Open the relevant section folder and open `index.html` (or the example file) in a browser to view demos.
- Solutions are included in `solution` folders or `solution.html` files where applicable.
- Assets (images, css) are kept inside each section’s folder.

Notes

- Files are organized by course section and lesson to mirror the course structure.
- I use this repo for experimentation, personal notes, and to store working examples.
